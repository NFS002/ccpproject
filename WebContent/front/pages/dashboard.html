Dashboard

This is the home page or `Dashboard' for a logged in user, Account information is fully loaded before the user reaches the page. If a user cannot log in, they cannot reach this page. This page provides several direct shortcuts to some of the more frequently used CCP user actions, such as writing an sending a new contract, as well as a recent notification panel, which is updated frequently without reloading the page, and shows information such as when a new contract or messsage is received. The navigation bar at the top of the page provide a link back to the brands main website by clicking their on its logo on the far left, or sign out of the current user session, and delete all currently loaded user data from the server, by clicking the 'Sign out' button on the right. The breadrumbs directly below this show the user what page s/he is on and anytime they find themselves on a different page, they can retrace their steps, or learn quickly how the intended route back to that page. The menu on the left provides a similar functionality, but it is more comprehensive, as any CCP intened for users page can be accessed directly from this menu. Some of the item on the meny ('Utilities' and 'Accounts') are two tiered, and when clicking on these items. a sub menu appears below. All other menu items are linked directly to another CCP user page. The search form at the top of this menu searches for keywords across all of the functionality provided within these pages.
